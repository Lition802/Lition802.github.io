今天上C语言课踩了个坑，我以为使用指针操作数组中的元素时，+1代表偏移一位，没想到是偏移一个内存单元。

``` c
int arr = {0,1,2,3}
```

对于这个数组，只需要下面的代码就可以进行遍历。

``` c
int *p;
p=arr; 指针指向数组首个元素
// 这里直接写死遍历4次
int i;
for(i=0;i<=4;i++){
    printf("%d",*p); // 在这里可以进行操作
    p++;
}
```
